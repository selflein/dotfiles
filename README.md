## Dotfiles for tiling window manager setup
### Required packages
* For regular X11/i3 setup
	* `i3-gaps`
	* `i3-lock`
	* `polybar`
	* `dunst`
	* `xautolock`
	* `autorandr`
	* `redshift`
	* `geoclue`
* For wayland/sway setup
	* `sway`
	* `waybar`
* Fonts
	* `San Francisco OTF`
	* `Source Code Pro OTF`
	* `Font Awesome 5`
	* `Material Icons`
